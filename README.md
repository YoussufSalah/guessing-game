# Guessing game

## How to use
- Open the terminal
- Run `git clone https://github.com/YoussufSalah/guessing-game.git/`
- Run `cd guessing-game`
- Run `node app.js`
- Select difficulty from the list ( 1 -> Easy, 2 -> Medium, 3 -> Hard )
- Enter your choises.
- Good luck!
